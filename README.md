# OpsCore

Unified service-lifecycle platform — alert management, incident management,
and change management in a single, modular hub.

## Status

Early development. Building the MVP foundation first:

1. Authentication & role-based access
2. Service Catalog (three-tier service model)
3. Alert Management (ingestion, live console, resolution)

Incident and Change modules follow once the core lifecycle is in place.

## Stack

- **Frontend:** React + Vite (TypeScript)
- **Backend:** Python + FastAPI
- **Database:** PostgreSQL
- **Cache / queue:** Redis
- **Search:** Elasticsearch
- **Realtime:** WebSockets
